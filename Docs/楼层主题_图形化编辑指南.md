# 楼层主题 · 图形化编辑指南

楼层显示名 / 进场旁白 / 相机背景色 / 背景纹理算法 全部数据化。无需改 C# 即可调主题。

---

## 入口

`Project` 面板进入 `Assets/Resources/Floors/`，双击：

- `Floor1_Inferno.asset` —— 第一层「炼狱」
- `Floor2_FrostRealm.asset` —— 第二层「冰霜」
- `Floor3_ChaosAbyss.asset` —— 第三层「虚空」

运行时通过 `Resources.LoadAll<FloorThemeData>("Floors")` 扫描，按 `floorNumber` 匹配当前层。

---

## Inspector 字段

| 字段 | 作用 |
|---|---|
| `Floor Number` | 楼层编号（1/2/3...）。`GameBootstrap` 按此匹配，必填 |
| `Display Name` | HUD 顶栏显示名（如「Inferno」「炼狱」） |
| `Narrative Banner` | 进入该层时显示的剧情横幅（多行可编辑） |
| `Camera Background` | 主相机背景色（远景填充） |
| `Procedural Kind` | 背景纹理算法：`None` / `Inferno` / `Frost` / `Chaos` / `Custom` |
| `Custom Background` | 当 `Procedural Kind = Custom` 时使用的预制 Sprite；其他模式忽略 |

### Procedural Kind 选项

| 值 | 效果 |
|---|---|
| `None` | 不生成背景（地图自带瓦片足够时） |
| `Inferno` | 深红岩石 + 多层岩浆裂纹 + 4 热点 |
| `Frost` | 深蓝底 + 六角晶格 + 雪花高光 + 立柱条纹 |
| `Chaos` | 极紫漩涡 + 中央虚空 + 高频混沌裂纹 |
| `Custom` | 用 `Custom Background` 字段拖入自己的精灵 |

---

## 容错

- 任意 `*.asset` 缺失 → `GameBootstrap` 回退到代码 switch（与旧行为完全一致）
- `floorNumber` 重复或不存在 → 同样回退
- 即使删除整个 `Floors/` 文件夹也不会崩溃

---

## 重置回默认

```
执行 Assets/Editor/Tools/CreateFloorThemesDefault.cs → Execute
```

会按代码版重新生成 3 个资产。

---

## 替换为你自己的图

1. Procedural Kind 改为 `Custom`
2. 把你的 PNG 拖入 `Custom Background` 字段
3. PNG 任意尺寸都行，`FloorBackground` 会按地图大小自动缩放
4. 建议 PPU = 20、Wrap = Clamp、Filter = Bilinear（按现行约定）
