# 商店视觉装饰 · 图形化编辑指南

商店的**老板像素小人 / 货架 / 武器陈列**已全部数据化。无需改 C# 代码即可改造商店外观。

---

## 入口

`Project` 面板进入 `Assets/Resources/Shop/`，双击 **`Default.asset`** → Inspector 即可编辑。

> 运行时通过 `Resources.Load("Shop/Default")` 加载。如果该资产被删除，`GameBootstrap.BuildShopRoom` 会自动回退到代码内置的程序化默认值，所以**修改不会破坏游戏**。

---

## Inspector 字段说明

### 商店老板 / 货架（`ShopDecorGroup`）

每组就是 Hierarchy 里的一个 GameObject。每组包含：

| 字段 | 作用 |
|---|---|
| `Name` | 调试用名字（生成的 GameObject 名） |
| `Anchor` | 该组在房间内的世界坐标（房间中心 = (0,0)） |
| `Enabled` | 取消勾选则不生成 |
| `Parts` | 像素方块列表（如下） |

每个 `Part`：

| 字段 | 作用 |
|---|---|
| `Note` | 调试名（如「兜帽」「眼睛-左」） |
| `Local Pos` | 相对 Anchor 的本地坐标（世界单位） |
| `Size` | 方块尺寸（世界单位） |
| `Color` | 颜色 |
| `Sorting Order` | 数字越大越靠前（用来盖在其他 part 上） |

### 武器陈列

| 字段 | 作用 |
|---|---|
| `Weapon Row Start` | 最左侧武器的世界坐标 |
| `Weapon Spacing` | 武器之间的水平间距（默认 2.2） |
| `Weapon Scale` | 每件武器精灵的缩放（默认 1.1） |
| `Weapon Sorting Order` | 武器精灵层级 |
| `Weapon Glow Sorting Order` | 武器下方稀有度发光底座层级 |
| `Weapon Glow Local Pos / Scale / Alpha` | 发光底座的位置、大小、透明度 |
| `Weapon Collider Radius` | 武器触发盒半径（玩家走近即拾取）|

武器实物精灵和颜色由现有 `WeaponSprites` / `WeaponData.GetRarityColor()` 自动决定。

### 额外陈设（`Extra Decor`）

想加灯笼、酒桶、挂画？不用碰代码，直接在 Inspector 加一个新的 `ShopDecorGroup` 到 `Extra Decor` 列表里，设好 Anchor + 几个 Part，运行就有。

---

## 重置回默认

如果调坏了，运行一次 Editor 脚本：

```
执行 Assets/Editor/Tools/CreateShopDecorDefault.cs → Execute
```

会重新生成 `Default.asset`，与代码版完全一致。
