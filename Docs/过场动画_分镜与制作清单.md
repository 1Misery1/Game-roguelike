# 过场动画 · 分镜与制作清单

本文给出**全部 8 段过场动画**的分镜规格：场景 / 出场角色 / 镜头节奏 / 台词 / 时长。
美术与动画师按此清单制作；接入到游戏的 hook 已留好，见末节「接入点」。

> 像素分辨率统一：256×144（与 `FloorBackground` 同制式）。PPU=20。
> 配色与已有的 Inferno / Frost / Chaos 主题保持一致。
> 所有台词全角中文，标点跟随简体习惯，引用句用「」。

## 通用规范

### 像素与色彩

- 分辨率：256×144（与 `FloorBackground` 相同）
- 帧率：12 fps（像素感更强；逐帧绘制可压到 6-8）
- 调色板：Inferno（深红橙）/ Frost（深蓝白）/ Chaos（暗紫品红）/ Crown（暖金红）— 跟现有 `FloorThemeData.cameraBackground` 系列对齐
- 字体：方舟像素体（已统一）

### 跳过 & 暂停

- 所有过场期间 `Time.timeScale = 0`（黑幕场景已实现）
- 玩家按 **空格 / 回车 / Esc / 鼠标左键** 可强制跳过 → 调用 `GameBootstrap.Instance.SkipEndingCutscene()`
- 不可跳过的过场：A 开场（首次启动）建议允许跳过；B-D Boss 战前可允许跳过

### 接入点

| 过场 | 触发位置 | 可用 hook |
|---|---|---|
| A. 开场 | MenuController「开始游戏」 | 需新增 `OnGameStart` 事件 |
| B. F1 Boss 战前 | `GameBootstrap.BuildFloor1Boss` 顶部 | 需新增 `OnBossIntro(int floor)` 事件 |
| C. F2 Boss 战前 | `GameBootstrap.BuildFloor2Boss` 顶部 | 同上 |
| D. F3 Boss 战前 | `GameBootstrap.BuildFloor3Boss` 顶部 | 同上 |
| E. 隐藏 Boss 战前 | `GameBootstrap.SpawnHiddenBoss` 内的 `DialogueBox.Play` 之前 | 可替换现有对话块 |
| F. 普通结局 | 已接入 `OnEndingCutsceneStart(false, 4.0f)` | ✅ 直接订阅 |
| G. 真相结局 | 已接入 `OnEndingCutsceneStart(true,  6.5f)` | ✅ 直接订阅 |
| H. 王冠结局 | 已接入 `OnEndingCutsceneStart(true,  8.5f)` | ✅ 直接订阅 + 用 `_endingTier == Crown` 判断 |

### 真实动画接入示例（结局类）

```csharp
GameBootstrap.OnEndingCutsceneStart += (isTrue, duration) =>
{
    var tier = GameBootstrap.Instance.CurrentEndingTier;   // 待暴露
    var clip = tier switch {
        GameBootstrap.EndingTier.Crown  => crownClip,
        GameBootstrap.EndingTier.Truth  => truthClip,
        _                                => normalClip,
    };
    myVideoPlayer.clip = clip;
    myVideoPlayer.Play();
    myVideoPlayer.loopPointReached += _ => GameBootstrap.Instance.SkipEndingCutscene();
};
```

### Boss 战前 / 开场接入（需补 hook）

实现时可在 GameBootstrap 添加：

```csharp
public static event System.Action<int> OnBossIntro;   // 参数：floor (1/2/3/4)
public static event System.Action OnGameStart;
```

`BuildFloorXBoss` / `MainMenu.StartGame` 在合适位置触发即可。
